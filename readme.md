# 🔓 ZIP Cracker Pro (Python)

A multi-strategy ZIP password recovery tool using brute-force, dictionary, and hybrid attacks.

---

## 🚀 Features

- Smart numeric attack (fast for PIN-style passwords)
- Dictionary attack (wordlist based)
- Hybrid attack (word + number)
- Pattern attack (dates like 0616, 2501)
- Parallel brute-force (multi-core CPU)
- Resume support (basic)

---

## 📦 Requirements

- Python 3.x

---

## ▶️ Usage

```bash
python3 zip_cracker_pro.py
